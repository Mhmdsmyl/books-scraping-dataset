# 📚 Books Scraping Dataset

This project is a simple web scraping exercise where I collected data from an online bookstore and built a structured dataset.

## 🚀 Project Overview
- Scraped around **1000 books**
- Extracted useful information such as:
  - Book Title
  - Price
  - Rating

## 🛠️ Tools & Technologies
- Python
- BeautifulSoup
- Pandas

## 📊 Dataset
The dataset contains clean and organized data that can be used for:
- Data analysis
- Visualization
- Practice projects

## 💡 What I Learned
- How to scrape data from websites using BeautifulSoup  
- Handling and cleaning real-world data  
- Structuring data using Pandas
